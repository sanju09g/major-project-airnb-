# Wanderlust - Airbnb Clone

📌 Project Overview

   Wanderlust is a responsive full-stack Airbnb clone built with Node.js and Express.js using the MVC architecture. It allows users to explore listings, create and manage their own stays, and leave reviews — all      while enforcing user authentication and authorization. The app features category-based filtering and Mapbox integration for interactive maps.
   
   
## 🚀 Features

  -  Clean and responsive UI for different screen sizes

  - User Authentication (Signup / Login / Logout)

  - Create, Edit,Update and Delete Listings
  
  - Upload one images per listing (Cloudinary + Multer)

  - Add and delete Reviews for listings

  - Authorization: Only owners can delete their listings or reviews

  - Mapbox integration to display listing locations

  - Category filtering and search

  - Routes Protection ( only authenticated users are able to access listings page )



 ## 🛠️ Technologies used

 - Node.js
  
 - Express.js

 - MongoDB + Mongoose
  
 - EJS + EJS Mate (Templating engine)
  
 - Passport.js (Authentication)
  
 - Cloudinary + Multer (Image Uploads)
  
 - Mapbox SDK
  
 - Joi (Request validation)
  
 - Method Override
  
 - Express Session + Connect Mongo
  
 - dotenv
